🎯 QUICK START IN 3 STEPS
═════════════════════════════

Step 1️⃣  Load Extension in Chrome
   1. Open chrome://extensions/
   2. Enable "Developer mode" (top right)
   3. Click "Load unpacked"
   4. Select the Autofill-Graph folder
   ✅ Extension should appear in your toolbar

Step 2️⃣  Configure API Key
   1. Get free key: https://console.mistral.ai/api-keys/
   2. Click extension icon in toolbar
   3. Go to "Settings" tab (bottom left)
   4. Paste your Mistral API key
   5. Click "Save API Key"
   ✅ You should see "✅ Connected" in the header

Step 3️⃣  Test Autofill
   1. Visit any website with a form (Gmail signup, GitHub, etc.)
   2. Click extension icon
   3. Click "🔍 Detect Forms"
   4. Click "✨ Autofill"
   ✅ Form fields should fill with sample data!

🧠 HOW IT WORKS
═════════════════════════════

1. USER FILLS FORM
   ↓
2. EXTENSION DETECTS FORM
   (content.js analyzes DOM)
   ↓
3. USER CLICKS AUTOFILL
   ↓
4. BACKGROUND SCRIPT QUERIES KNOWLEDGE GRAPH
   (background.js → knowledgeGraphManager)
   ↓
5. VECTOR DB FINDS SIMILAR ENTITIES
   (cosine similarity matching)
   ↓
6. MISTRAL API GENERATES RESPONSES
   (entity extraction & narrative generation)
   ↓
7. CONTENT SCRIPT INJECTS VALUES
   (form fields fill automatically)
   ↓
8. USER REVIEWS & EDITS
   ↓
9. USER CLICKS "LEARN THIS FORM"
   ↓
10. KNOWLEDGE GRAPH UPDATES
    (new entities & relationships added)
   ↓
11. NEXT FORM AUTOFILLS BETTER! ✨

❓ TROUBLESHOOTING
═════════════════════════════

API Key Not Working?
  → Check that you copied the full key from console.mistral.ai
  → Try pasting again with no extra spaces

Forms Not Detected?
  → Some websites use custom form frameworks
  → Try a simple website first (Google Forms, GitHub, etc.)

Autofill Returns Nothing?
  → Make sure sample data loaded (check Graph Status)
  → Try clicking "Detect Forms" first
  → Check browser console (F12) for errors

Still Stuck?
  → See TESTING_GUIDE.md → Troubleshooting section
  → Check console logs: F12 → Console tab
